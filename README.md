# Smart Faculty Dashboard

Smart Faculty Dashboard is a React + TypeScript web app for faculty and students to manage subjects, reviews, submissions, messaging, analytics, leaderboards, and AI-assisted feedback in one place.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Router
- shadcn/ui

## Getting Started

```sh
npm install
npm run dev
```

## Production Build

```sh
npm run build
```

## Main Features

- Faculty dashboard with branded navigation and profile flows
- Subject management and routed subject pages
- Notification center with outside-click handling and animation
- Hero slider with working controls and Framer Motion transitions
- DiceBear-powered user avatars with fallback initials
- Real file uploads with drag and drop, validation, and progress
- AI feedback generation with loading and regeneration support
- Leaderboard and analytics views
- Light and dark theme support with improved contrast
